### Reflection

* Did the topics discussed in your labs prepare you to complete the assignment? Be specific, which features in your weekly assignment did you feel unprepared to complete?

For the most part yeah. The labs covered the basics I needed like the box model, flexbox, and media queries, and that was most of the project. The part I felt least ready for was the responsive nav. We never did a mobile hamburger menu in the labs so I had to figure out the CSS only checkbox version on my own. Getting the CSS Grid sections to behave at every screen size also took way more trial and error than I thought it would, especially the borders at the medium tablet sizes.

* If you had more time, what would you have done differently? Would you have added additional features? Changed the way your project responded to a particular event, etc.

If I had more time I would actually build the Turkish embroidery / Iznik tile divider that I put in my design spec, since I ran out of time and never added it. I would also fix the fact that my nav and footer are copy pasted across all four pages, that got annoying to keep in sync every time I changed something. Maybe add a real second guide page on top of the food one too.

* Reflect on your project demo, what went well? Were there things that maybe didn't go as planned? Did you notice something that your peer did that you would like to try next time?

The neobrutalist style came together better than I expected and people liked the whole cats running Istanbul angle for the gallery. What didnt totally go to plan was the responsiveness. I kept finding little border and spacing bugs at medium screen widths that I had to chase down near the end. Next time I want to test on actual phone widths way earlier instead of saving it for last. One peer had a really clean sticky section setup that I thought looked nice and want to try out later.
